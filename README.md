# IP Services Knowledge Base

A searchable, single-file HTML knowledge base covering:

- **IP Services (general)** — full lifecycle by matter type (patent, design, trademark, copyright), filing, recordals, renewals & annuities, other IP rights, IPMS/portal systems, people & roles, providers & market, and cross-cutting comparisons.
- **RWS PatOrg Knowledge Base** — a dedicated section (switchable via the mode toggle in the sidebar) covering the RWS IP Portal Digitalisation Programme, PatOrg getting-started/navigation, Recordals processes (project creation, country folders, workflows, due dates, documents, invoicing), the MVP Portal end-to-end process, and release notes.

No build step — it's a static HTML file with inline CSS/JS. To update, edit `index.html` directly and push.
